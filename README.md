# Roblox Username Checker

System to check if a username is valid or invalid.

Could you fork it on GitHub, follow it, or give it a star, please!

---

## Installation

```bash
git clone https://github.com/pqpcara/roblox-username.git
cd roblox-username
```

---

## How to use

### NPM Installation

```bash
npm install
```

### Build Developer Mode

```bash
npm run build
```

### Start Developer Mode

```bash
npm run dev
```

### Start System

```bash
npm run start
```
